**Review of Univariate Linear Model Analysis**

* The main goal of the linear model is to evaluate relationships in order to explain variability in a response
variable as a function of some specified model and an error of prediction:

.. math::

   \text{Response = Model + Error}

* Univariate regression models can be expressed mathematically as a regression function,

,, math::

   Y = \beta_0 + \beta_1 X_1 + \epsilon

for a simple model with a single predictor variable. 

* For a more complex model with multiple predictors, we may write

.. math::

   Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + ... + \beta_q X_q + \epsilon
