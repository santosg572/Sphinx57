Ejercicios
==========

URL: https://es.wikipedia.org/wiki/Serie_de_Taylor


Calcular las sigioentes sumas:

1. :math:`S_{100} = \sum_{n=0}^{100} \frac{1}{2^n}`

2. :math:`S_{100} = \sum_{n=0}^{100} \frac{1}{n}`

3. :math:`S_{100} = \sum_{n=0}^{100} (-1)^{n+1}\frac{1}{n}`

4. Para x = 1 y x=2. Calcular

.. math::

   S = \sum_{n=1}^{\infty} \frac{(-1)^{n+1}}{n} x^n

5. Para x=2 y x = -2, calcular:

.. math::

   S = \sum_{n=2}^{100} \frac{n(n-1)}{2} x^{n-2}
